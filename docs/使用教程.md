# Options Level Pro 使用教程 / Options Level Pro User Guide

Options Level Pro 是 Trading Hub 的 ATAS 指标，用于在图表上绘制期权关键位水平线。当前正式版本为 `1.0.0`。

Options Level Pro is a Trading Hub ATAS indicator that draws options key levels as horizontal chart levels. The current stable version is `1.0.0`.

## 下载 / Download

请根据 ATAS 大版本下载对应 DLL：

Download the DLL that matches your ATAS major version:

```text
Options-Level-Pro-1.0.0-ATAS-7.x.dll
Options-Level-Pro-1.0.0-ATAS-8.x.dll
```

ATAS 7.x 和 ATAS 8.x 的 DLL 不能混用。

DLLs for ATAS 7.x and ATAS 8.x are not interchangeable.

## 安装 / Installation

1. 关闭 ATAS。
   Close ATAS.
2. 将 DLL 复制到 ATAS Indicators 目录。
   Copy the DLL into the ATAS Indicators folder.

```text
%AppData%\ATAS\Indicators\
```

3. 重新启动 ATAS。
   Restart ATAS.
4. 打开图表并添加 Options Level Pro 指标。
   Open a chart and add the Options Level Pro indicator.
5. 在指标设置中填写 License Key。
   Enter your License Key in the indicator settings.

## 使用说明 / Usage Notes

- 指标会在图表上绘制当前产品支持的期权关键位水平线。
  The indicator draws supported options key levels as horizontal chart levels.
- 请确认图表品种和数据周期符合你的交易计划。
  Make sure the chart symbol and timeframe match your trading plan.
- 如需更换设备，请联系管理员重置机器绑定后重新激活。
  To move to another device, ask the administrator to reset the machine binding before activating again.

## 风险提示 / Risk Notice

Options Level Pro 只显示辅助参考水平，不提供交易信号、投资建议或盈利保证。实盘使用前必须先模拟测试。

Options Level Pro only displays reference levels. It does not provide trading signals, investment advice, or any profit guarantee. Test in simulation before live trading.
