# Options Level Pro 用户版更新日志 / Options Level Pro User Changelog

本文记录公开下载镜像中的用户可见版本变更。

This file records user-visible changes in the public download mirror.

## 0.6.1

发布日期：正式发布。

Release date: stable release.

### 更新 / Changed

- 更新 GC / MGC 关键位绘制逻辑。
  Updated GC / MGC key-level drawing behavior.
- GC / MGC 的 Call OI 和 Put OI 标签统一显示为 OI。
  Unified GC / MGC Call OI and Put OI labels as OI.
- 提供 ATAS 7.x 和 ATAS 8.x 两个正式版 DLL。
  Provides stable DLLs for both ATAS 7.x and ATAS 8.x.

## 0.6.0

发布日期：正式发布。

Release date: stable release.

### 新增 / Added

- 提供 ATAS 7.x 专用 DLL。
  Added a dedicated DLL for ATAS 7.x.
- 提供 ATAS 8.x 专用 DLL。
  Added a dedicated DLL for ATAS 8.x.
- 支持期权关键位水平线绘制。
  Added options key-level horizontal line rendering.

### 注意 / Notes

- 请根据自己的 ATAS 大版本下载对应 DLL。
  Download the DLL that matches your ATAS major version.
