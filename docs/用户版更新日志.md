# Options Level Pro 用户版更新日志 / Options Level Pro User Changelog

本文记录公开下载镜像中的用户可见版本变更。

## 1.0.0

发布日期：正式发布。
Release date: stable release.

### 修复 / Fixed

- 合并本地历史缓存为 `options_level_pro_history.json`，NQ、GC 等品种历史数据会按品种分组保存。
  Merged local history cache into `options_level_pro_history.json`, with NQ, GC, and other symbol histories grouped by symbol.
- 首次加载会迁移旧的 `options_level_pro_history_NQ.json`、`options_level_pro_history_GC.json` 等分散历史文件。
  First load migrates legacy split files such as `options_level_pro_history_NQ.json` and `options_level_pro_history_GC.json`.
- 修复历史 VolTrig、Combo、Large Gamma 等关键位的颜色和显示恢复。
  Fixed historical VolTrig, Combo, Large Gamma, and related level color and visibility restore.
This file records user-visible changes in the public download mirror.

## 0.6.2

发布日期：正式发布。

Release date: stable release.

### 修复 / Fixed

- 修复加载历史记录时只显示 Call Wall、Put Wall 和 Zero Gamma 的问题，现在会恢复 Vol Trigger、Combo、Large Gamma 等完整关键位。
  Fixed historical loading so Vol Trigger, Combo, Large Gamma, and other key levels are restored instead of only Call Wall, Put Wall, and Zero Gamma.
- 保留 `0.6.1` 作为历史版本，`0.6.2` 曾作为推荐下载版本。
  Preserved `0.6.1` as a historical release; `0.6.2` was the recommended download for that release window.

## 0.6.1

发布日期：正式发布。

Release date: stable release.

### 更新 / Changed

- 更新 GC / MGC 关键位绘制逻辑。
  Updated GC / MGC key-level drawing behavior.
- GC / MGC 的 Call OI 和 Put OI 标签统一显示为 OI。
  Unified GC / MGC Call OI and Put OI labels as OI.
- 提供 ATAS 7.x 和 ATAS 8.x 两个正式版 DLL。
  Provides stable DLLs for both ATAS 7.x and ATAS 8.x.

## 0.6.0

发布日期：正式发布。

Release date: stable release.

### 新增 / Added

- 提供 ATAS 7.x 专用 DLL。
  Added a dedicated DLL for ATAS 7.x.
- 提供 ATAS 8.x 专用 DLL。
  Added a dedicated DLL for ATAS 8.x.
- 支持期权关键位水平线绘制。
  Added options key-level horizontal line rendering.

### 注意 / Notes

- 请根据自己的 ATAS 大版本下载对应 DLL。
  Download the DLL that matches your ATAS major version.
