# Options Level Pro 用户版更新日志 / Options Level Pro User Changelog

本文记录公开下载镜像中的用户可见版本变更。

This file records user-visible changes in the public download mirror.

## 1.0.0

发布日期：正式发布。

Release date: stable release.

### 更新 / Changed

- 重新建立历史缓存逻辑，ATAS 7.x / 8.x 正式版统一使用 `options_level_pro_history.json` 全量历史文件，不再依赖 `options_level_pro_history_NQ.json`。
  Rebuilt the history-cache logic for ATAS 7.x / 8.x stable builds to use the full-data `options_level_pro_history.json` file instead of `options_level_pro_history_NQ.json`.
- 写回本地历史缓存时只更新当前品种的 `symbols.{symbol}.history`，保留同一文件内其它品种数据。
  Local history-cache writes update only `symbols.{symbol}.history` for the current symbol and preserve other symbols in the same file.
- 保留 `0.6.2` 作为历史版本，当前推荐下载为 `1.0.0`。
  Preserved `0.6.2` as a historical release; the current recommended download is `1.0.0`.

## 0.6.2

发布日期：正式发布。

Release date: stable release.

### 修复 / Fixed

- 修复加载历史记录时只显示 Call Wall、Put Wall 和 Zero Gamma 的问题，现在会恢复 Vol Trigger、Combo、Large Gamma 等完整关键位。
  Fixed historical loading so Vol Trigger, Combo, Large Gamma, and other key levels are restored instead of only Call Wall, Put Wall, and Zero Gamma.
- 保留 `0.6.1` 作为历史版本，`0.6.2` 是当时的推荐下载版本。
  Preserved `0.6.1` as a historical release; `0.6.2` was the recommended download for that release cycle.

## 0.6.1

发布日期：正式发布。

Release date: stable release.

### 更新 / Changed

- 更新 GC / MGC 关键位绘制逻辑。
  Updated GC / MGC key-level drawing behavior.
- GC / MGC 的 Call OI 和 Put OI 标签统一显示为 OI。
  Unified GC / MGC Call OI and Put OI labels as OI.
- 提供 ATAS 7.x 和 ATAS 8.x 两个正式版 DLL。
  Provides stable DLLs for both ATAS 7.x and ATAS 8.x.

## 0.6.0

发布日期：正式发布。

Release date: stable release.

### 新增 / Added

- 提供 ATAS 7.x 专用 DLL。
  Added a dedicated DLL for ATAS 7.x.
- 提供 ATAS 8.x 专用 DLL。
  Added a dedicated DLL for ATAS 8.x.
- 支持期权关键位水平线绘制。
  Added options key-level horizontal line rendering.

### 注意 / Notes

- 请根据自己的 ATAS 大版本下载对应 DLL。
  Download the DLL that matches your ATAS major version.
