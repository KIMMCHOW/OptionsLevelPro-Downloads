# Options Level Pro 用户版更新日志 / Options Level Pro User Changelog

本文记录公开下载镜像中的用户可见版本变更。

This file records user-visible changes in the public download mirror.

## 1.0.0

发布日期：正式发布。

Release date: stable release.

### 修复 / Fixed

- 修复 ATAS 7.x / 8.x / ATAS X 图表偶发显示 `0.00` 水平线的问题，确保指标只绘制有效期权关键位。
  Fixed the occasional `0.00` horizontal line on ATAS 7.x / 8.x / ATAS X charts so the indicator only draws valid options key levels.
- 修正 ATAS X 下载包版本号，统一为 `1.0.0`，不再使用旧的错误路径或文件名。
  Corrected the ATAS X download-package version number to `1.0.0` and removed the old incorrect paths and filenames.

### 更新 / Changed

- 重新建立历史缓存逻辑，ATAS 7.x / 8.x / ATAS X 正式版统一使用 `options_level_pro_history.json` 全量历史文件，不再依赖 `options_level_pro_history_NQ.json`。
  Rebuilt the history-cache logic for ATAS 7.x / 8.x / ATAS X stable builds to use the full-data `options_level_pro_history.json` file instead of `options_level_pro_history_NQ.json`.
- 写回本地历史缓存时只更新当前品种的 `symbols.{symbol}.history`，保留同一文件内其它品种数据。
  Local history-cache writes update only `symbols.{symbol}.history` for the current symbol and preserve other symbols in the same file.
- 当前公开下载仓库只保留 `1.0.0`；旧 `0.6.x` 下载包已移除。
  The public download mirror now retains only `1.0.0`; old `0.6.x` download packages have been removed.
