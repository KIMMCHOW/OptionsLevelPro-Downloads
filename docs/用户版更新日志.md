# Options Level Pro 用户版更新日志 / Options Level Pro User Changelog

本文记录公开下载镜像中的用户可见版本变更。内部构建、服务器运维和未公开测试流程不在这里展开。

This file records user-visible changes in the public download mirror. Internal builds, server operations, and non-public test procedures are intentionally not covered here.

## 0.6.0

发布日期：正式发布。

Release date: stable release.

### 新增 / Added

- 提供 ATAS 7.x 专用 DLL。
  Added a dedicated DLL for ATAS 7.x.
- 提供 ATAS 8.x 专用 DLL。
  Added a dedicated DLL for ATAS 8.x.
- 支持期权关键位水平线绘制。
  Added options key-level horizontal line rendering.
- 支持服务器端信号数据读取。
  Added server-side signal data retrieval.
- 支持 License Key、设备标识和 data token 请求链路。
  Added the License Key, device identifier, and data-token request flow.

### 注意 / Notes

- 当前版本为正式版，适合当前用户下载和实机验证。
  This build is the current stable release for user download and real-machine validation.
- 正式 DLL 已通过 Eziriz .NET Reactor 保护，文件名不带 `protected` 字样。
  Formal DLLs are protected with Eziriz .NET Reactor. The filenames intentionally do not include `protected`.
