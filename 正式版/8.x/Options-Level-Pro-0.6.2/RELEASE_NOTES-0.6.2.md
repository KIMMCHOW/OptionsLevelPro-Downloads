# 发布说明 / Release Notes / Options Level Pro 0.6.2 / ATAS 8.x

这是适用于 ATAS 8.x 的 Options Level Pro 正式发布包。

This is a stable Options Level Pro release package for ATAS 8.x.

## 修复 / Fixed

- 修复加载历史记录时只显示 Call Wall、Put Wall 和 Zero Gamma 的问题。
  Fixed historical loading that restored only Call Wall, Put Wall, and Zero Gamma.
- 历史记录现在会恢复 Vol Trigger、Combo、Large Gamma 等完整关键位。
  Historical records now restore Vol Trigger, Combo, Large Gamma, and the full key-level set.

## 注意 / Note

请下载与你 ATAS 大版本一致的 DLL，不要跨 ATAS 大版本混用。

Download the DLL that matches your ATAS version. Do not mix DLLs across ATAS versions.
